# 🏀 Basketball Scoreboard

A simple and interactive basketball scoreboard built with HTML, CSS, and vanilla JavaScript. This project was created to practice fundamental front-end development skills, including DOM manipulation, event handling, and creating a responsive layout.

## ✨ Features

- **Score Tracking:** Add +1, +2, or +3 points for both the Home and Guest teams.
- **Game Clock:** A functional countdown timer that can be started and stopped.
- **Timer Controls:** Interactive buttons to Start, Stop (pause), and Reset the game clock.
- **Full Reset:** A "Reset" button that functions as a "New Game" button, resetting both scores and the timer.
- **Responsive Design:** The layout seamlessly adapts for a great user experience on both desktop and mobile devices.
- **Glassmorphism UI:** A modern, frosted-glass effect on the scoreboard for a cool visual style.

## 🛠️ Technologies Used

-   **HTML5**
-   **CSS3**
    -   Flexbox for layout
    -   Media Queries for responsiveness
    -   Custom Fonts (`@font-face`)
-   **Vanilla JavaScript (ES6)**
    -   DOM Manipulation
    -   Event Listeners
    -   `setInterval` for the timer logic
